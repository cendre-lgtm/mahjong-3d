🀄 新春喜慶版：台日雙制立體麻將

這是一個基於 Three.js 開發的 3D 麻將遊戲，專為手機與平板操作優化，支援日本麻將與臺灣麻將雙規則。

🌟 遊戲特色

雙規則切換：自由選擇日麻（13張）或台麻（16張）規則。

響應式 UI：完美適配手機直立、橫向、平板及桌機螢幕。

3D 立體視角：支援單指旋轉桌面與自動鏡頭追蹤。

智慧操作：支援「選牌預覽」與「特寫點擊出牌」，專為小螢幕設計。

節慶氛圍：內建新春 BGM 與紅金喜慶配色。

代幣系統：模擬真實對局，具備開局扣款與胡牌獎勵。

🎮 如何開始

直接在瀏覽器開啟本專案的 GitHub Pages 連結。

選擇規則與模式（支援 1~4 人同機對戰）。

點擊「開局贏紅包」即可開始對局！

🛠️ 技術開發

渲染引擎：Three.js (WebGL)

動畫系統：TWEEN.js

介面框架：Tailwind CSS

開發環境：Google Gemini Canvas
